# webdistillation
This is released for public use to the world as a way to allow normal people with limited hardware to break into machine learning. 
It is provided for free, feel free to sponsor or hire me. Any problems fix yourself then feel free to let me know.
Currently supports GPT2 architecture, should be fairly adjustable to other models. 


Quick rundown
Download DistilGPT2 model from huggingface
Save DistilGPT2 files to Models folder in repo 
Be sure to put the Onnx in its own Onnx folder or you'll get errors.
NPM install (Will pop up please include "x" in dependencies) 
NPM install "x" Package sharp relies on I forgot the name of or to include in dependencies 
Get ANY text corpus 
Edit this line in converged_pipe_2.js      const corpusPath = process.argv[2] || './10-sonnets.txt'; To point to your own Corpus 
run chat node unified_chat_fixed.js ./unified-aether.json
